## 简介

[vue-element-admin]主要实现web端一些基础的业务代码，基于 [vue](https://github.com/vuejs/vue) 和 [element-ui](https://github.com/ElemeFE/element)实现。

## 功能

- 登录

- 列表
  - 分页
  - tab列表
  - 搜索列表

- 详情

- 表单

- 轮播

- 组件缓存（`keep-alive`向前清除缓存、返回不清除缓存）首页->列表->详情：清除缓存  详情->列表：不清除缓存
